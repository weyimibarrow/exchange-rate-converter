#Convrrrt, the EXCHANGE RATE CONVERTER

Convrrrt (aka ineedtochangemymoneyrightthehellnow.com) is a command line (for now) exchange rate converter that can convert to and from 40+ currencies. 

## Installation

Since this tool only works in the command line right now, these are the steps to use it:
1. Clone this repository
Do this by clicking on the green "Code" button next to the "About" section and copying the HTTPS link. 
2. Open your terminal and install requests
Then, open your Terminal and navigate to the desired folder that you'd like to continue with the install using cd. E.g:

```bash
cd Desktop/projects
```
From there, type in the following to install "requests"

```bash
pip install requests
```
3. git clone
Once requests is installed, type in git clone and paste in the link from the first step
```bash
git clone https://github.com/yourusername/coolrepo1
```
4. Run the program
Type the following into your terminal:

python main.py
